Juego de Memoria con Señales de Tráfico

Objetivo: Emparejar todas las señales de tráfico iguales.

Reglas:

  • El juego comienza con un tablero de cartas boca abajo.
  • Cada carta tiene una señal de tráfico en la parte posterior.
  • Las señales están en pares (por ejemplo, dos cartas con la señal de "Stop", dos con la señal de "Cruce Peatonal", etc.).

Cómo Jugar:

  • Haz clic en una carta para voltearla.
  • Haz clic en una segunda carta para voltearla también.
  • Si las cartas tienen la misma señal de tráfico, las cartas permanecen boca arriba.
  • Si las cartas no coinciden, se vuelven a voltear boca abajo.
  • El juego continúa hasta que todas las cartas hayan sido emparejadas.

Aspectos Educativos:

Cada vez que se emparejan dos cartas, se muestra información adicional sobre la señal de trafico.
